Documentation of Modlist

The modlist keeps its database in a JSON file. This is an example database:

[
  {
    "name":"Glowstone Wire",
    "other":"",
    "link":"http://www.minecraftforum.net/topic/944547-",
    "desc":"Aesthetic Place-able glowstone wires",
    "author":"GrygrFlzr",
    "type":"Universal",
    "required":[
      "Forge"
    ]
  },
  {
    "name":"Dummy Mod",
    "other":"hi",
    "link":"http://modlist.mcf.li/",
    "desc":"A test mod",
    "author":"ZeroLevels",
    "type":"Client",
    "required":[
      "Modloader"
    ]
  }
]

Fields name, other, link, desc and author can contain any string.
However, type can only hold one the following values:

  • Universal
  • SSP
  • SMP
  • LAN
  • Client
  • Server

The required array can either be empty (treated as No Dependency) or hold one or more of the following values:
  • Modloader
  • Forge Compatible
  • Forge Required
