  • Ingame Mod List

The Ingame Mod List provides the lists found on this site to you at any time while you're playing Minecraft. Simply press F12 (configurable) to open its GUI. While you are browsing a list, you can type to search. Simply begin typing, and the search bar will appear automagically.

This mod was coded and designed by Vazkii. Please show him support by downloading with the adf.ly link. Otherwise, use the direct link. The file is hosted on ZeroLevels' SugarSync.

For more information, or to report bugs, see Vazkii's Minecraft Forums thread for it here.

The Ingame Mod List is currently for versions 1.5 and up. It requires Minecraft Forge (downloads here)to be installed in order to run. Simply place IGML2 into your .minecraft/mods directory.
