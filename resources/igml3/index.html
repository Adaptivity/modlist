  • Ingame Mod List

The Ingame Mod List provides the lists found on this site to you at any time while you're playing Minecraft. Simply press F12 (configurable) to open its GUI. While you are browsing a list, you can type to search. To search by a specific version, type it into the version search bar. You can specify other criteria and sort the list using the radio buttons and checkboxes.

This mod was coded and designed by Duke605. Please show him support by downloading with the adf.ly link. Otherwise, use the direct link. The file is hosted on ZeroLevels' SugarSync.

The Ingame Mod List is currently for versions 1.6 and up. It requires Minecraft Forge (downloads here)to be installed in order to run. Simply place IGML3 into your .minecraft/mods directory.
